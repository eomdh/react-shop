## 🚀 학습 내용
- 전역상태관리를 위해 Redux 세팅 및 활용
- useSelector, useDispatch 훅 학습 
- 리액트 라우터를 활용하여 라우팅
- React.lazy 활용
- axios를 통해 서버와 데이터를 요청 및 응답
- CSS 프레임워크로 Bootstrap을 사용
- Styled-Component 문법을 학습
- SASS 활용
